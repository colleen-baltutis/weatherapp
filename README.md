Setup
---

Install Node.js if you haven't done so already.

NPM version check to verify install with

```
npm -v
```

Afterwards, open command line in this directory and run

```
npm install
```



Usage
---

1. `npm start`

2. Open [http://localhost:3000/](http://localhost:3000/).


Tests
---

```
npm test
```
